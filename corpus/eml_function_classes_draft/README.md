# MachLib EML Function-Class Draft Corpus

This corpus is OBSERVATION-tier and DRAFT_INTERNAL. It explores how D-finite, analytic, smooth, continuous, and boundary/non-example records can fit MachLib's EML style without adding Mathlib dependency and without making public theorem, proof, open-problem, release, upload, or package claims.

The records are symbolic research scaffolding only. They are not Lean theorem files, not full real-analysis formalizations, and not a claim that MachLib replaces the external formal-library ecosystem.

Guardrails for every record:

- `public_ready`: false
- `upload_allowed`: false
- `release_ready`: false
- `mathlib_dependency`: false
- `forge_compiler_change_required`: false
- `hardware_required`: false
- `status`: `DRAFT_INTERNAL`
