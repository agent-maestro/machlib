# MachLib Evidence Workbench service plan - 2026-05-20

## Service

Monogate Evidence Workbench service.

## Target user

An internal operator or technical reviewer preparing a private review decision.

## First MVP

A local CLI/service wrapper that emits a review packet from committed MachLib artifacts. The MVP does not deploy, publish, upload, or create release artifacts.

## Inputs

- zero-Mathlib gate output
- six-lane EML dashboard
- function-class frontier rollup
- stochastic/hybrid frontier rollup
- package candidate matrix
- phase spine
- private review branch status
- Command Center internal card/feed snapshots

## Outputs

- local readiness summary
- guardrail status
- recommended next task
- private review branch posture
- no-go boundary checklist

## Revenue path

Internal dogfood first, then AI-generated research/code/formal artifact review service.
