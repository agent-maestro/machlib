# MachLib Evidence Reel Reframe To Senses

Date: 2026-05-21

## Summary

The recent Evidence Reel work was a valuable misunderstanding. It did not land on the user's intended audio/visual product direction, but it produced reusable infrastructure that should be kept:

- storyboard generation
- SVG card rendering
- static HTML player
- Command Center static snapshot planning

These pieces are useful as local visual infrastructure, future Command Center card infrastructure, and future 1op/Senses visual card infrastructure.

## Reframe

The Evidence Reel should not be treated as the main AV product direction. The primary direction is 1op.io/senses: an interactive sensory layer around animal senses and alternate sensing modes.

Report videos and static status-only reels are supporting artifacts, not the destination.

## Senses Direction

The next product direction should focus on interactive models for:

- seeing
- hearing
- smelling
- touching
- electro-sensory perception
- magnetic-sensory perception
- infrared perception

The existing card renderer and HTML player can become early building blocks for sensory cards, local demos, and Command Center-mounted static snapshots.
